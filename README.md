🚀 Power BI Project: Pizza Sales Analytics Dashboard

I developed an interactive Power BI dashboard using a real-world pizza sales dataset to analyze customer behavior, product performance,and revenue trends,sales trends.

🔄 Data Preparation (ETL):
The dataset contained missing values and duplicate records. To ensure data accuracy:
• Performed data extraction and validation using SQL for accurate KPI calculations  
• Applied data cleaning and transformation using Power Query (handled null values, removed duplicates, and structured data for analysis)  

📊 Dashboard Highlights:
• KPI tracking: Total Revenue, Total Orders, Average Order Value, Average Pizzas per Order, and Total Pizzas Sold using DAX  
• Weekly and monthly order trend analysis using bar and line charts  
• Category-wise revenue contribution (%) using donut charts  
• Funnel visualization for total pizzas sold by category  
• Top 5 and Bottom 5 products analysis based on revenue, orders, and quantity  

📈 Key Business Insights:
• Peak demand observed during weekends, especially Friday and Saturday evenings  
• Seasonal spikes identified in January and July  
• Large-sized pizzas are the most preferred by customers  
• Classic Deluxe category contributes the highest sales volume  
• Brie Carre category shows the lowest performance in terms of quantity sold  

🛠️ Tools & Technologies:
• SQL (Data Analysis & Kpi's)  
• Power BI  
• Power Query (Data Cleaning & Transformation)  
• DAX (Data Analysis Expressions)  

This project demonstrates my skills to work with real-world datasets, perform end-to-end data analysis, and generate actionable insights for business decision-making.
